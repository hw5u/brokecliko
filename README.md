# Broke Cliko II

Broke Cliko II is the sequel to the original Broke Cliko, that I made in late 2025.
If you want to try the first game, you can play it here: [Broke Cliko](https://jeekwol.itch.io/broke-cliko)
*(Warning: it may contain some disrespectful or vulgar content.)*

As the name suggests, Broke Cliko is a clicker game where you start broke (like broke boi boi) and buy upgrades to become richer in the long term.

For now, the game is still in early development and is not fully playable. There is no executable (.exe) version yet, so if you want to try it, follow the instructions below.

# Installation

Currently only available Windows instructions

## Windows

### 1. Download LÖVE

LÖVE (also called LOVE2D) is a framework used to create 2D games with Lua.
Download it from the official site: https://love2d.org/

During installation, make sure **"Do not create shortcuts" is unchecked**.

---

### 2. Download Broke Cliko II project

You don’t need another website:

1. Scroll up to the top of this page
2. Click the **Code** button
3. Click **Download ZIP**
4. Extract the ZIP file on your desktop

---

### 3. Run the game

1. Make sure LÖVE is installed
2. Drag the extracted project folder onto the LÖVE shortcut on your desktop
